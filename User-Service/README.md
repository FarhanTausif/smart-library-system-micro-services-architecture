# User Service

Handles registration, profile management, and user-related queries.

- REST Base Path: /api/users
- Database: user_db (table: users)

## Development

1. Install dependencies:
   npm install
2. Start service:
   npm start

## Endpoints
- GET /api/users
- GET /api/users/:id
- POST /api/users
